# Quick Start Guide

## TL;DR - Get Started in 5 Minutes

### 1. Install & Build (2 minutes)
```bash
npm install
npm run build
```

### 2. Create Icons (1 minute)
**Easiest method:**
- Visit https://redketchup.io/icon-converter
- Upload `/public/icon.svg`
- Generate PNG at sizes: 16, 48, 128
- Download and save as `icon16.png`, `icon48.png`, `icon128.png` in `/public/`
- Run `npm run build` again

### 3. Load in Chrome (1 minute)
1. Open `chrome://extensions/`
2. Enable "Developer mode" (top-right toggle)
3. Click "Load unpacked"
4. Select the `dist` folder
5. Done! âœ…

### 4. Configure & Use (1 minute)
1. Click extension icon in Chrome toolbar
2. Fill in your info
3. Click "Save Settings"
4. Visit a job application page
5. Click "Auto-Fill Current Page"

## Common Questions

**Q: Will this work on all job sites?**  
A: It works on most sites, but some custom forms may need manual filling.

**Q: Is my data safe?**  
A: Yes! Everything is stored locally in Chrome. No data is sent anywhere.

**Q: Can I use this on multiple computers?**  
A: Yes! Export your data from one computer and import on another.

**Q: What if a field doesn't fill correctly?**  
A: You can always manually adjust any field after auto-fill.
